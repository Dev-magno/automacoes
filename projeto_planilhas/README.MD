
# Relatório de Clientes Inadiplentes

Este projeto em python automatiza a análise de uma planilha simples de clientes e gera um relatório  com os inadiplentes.

## Funcionalidades
- Leitura de planilha de Excel
- Filtragem de clientes com saldo negativo
- Geração de novo arquivo Excel com os resultados


## Bibliotecas usadas
- pandas
- openpyxl
- os

## Como usar
1. Substitua ```clientes.xlsx``` pela sua planilha
2. Execute o script
3. O arquivo ```relatorio_inadiplentes.xlsx``` será gerado automaticamente


#### Ideal para empresas que desejam otimizar processos administrativos com automação simples e eficaz.
